### 3. Automatic Tag Injection

**Function:** `dozo_inject_missing_tags($base_path)`

**Process:**

- Scans all HTML, CSS, JS files recursively
- Checks for `@dozo:sync` tag
- Injects appropriate tag if missing:
  - HTML: `<!-- @dozo:sync auto -->`
  - CSS: `/* @dozo:sync auto */`
  - JS: `// @dozo:sync auto`

**Example Injection:**

**Before:**

```html
<div class="warranty-form">...</div>
```

**After:**

```html

```

**Implementation:**

```php
add_action('init', function() {
    if (WP_DEBUG && WP_DEBUG_LOG) {
        $last_run = get_option('dozo_smartsync_last_run', 0);
        if (time() - $last_run > 86400) { // 24 hours
            dozo_smartsync_validate_layout();
            update_option('dozo_smartsync_last_run', time());
```
